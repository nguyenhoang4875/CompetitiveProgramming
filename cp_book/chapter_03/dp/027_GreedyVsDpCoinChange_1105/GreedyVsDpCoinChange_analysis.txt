APPROACH:
- [Algorithm or method]
    Ta cần tìm số lượng đồng có tổng là x với cách greedy và dp
    giả sử x là số tiền nhỏ nhất mà greedy(x) > opt(x);
    gọi y < x thì: greedy(x) > opt(x)
    nếu x >= a[n - 1] + a[n - 2] => x = q * c[n] + r;
    vì r < x nên greedy(x) = q + opt(r);
    Bằng phép trao đổi có thể thay 1 đồng a[n - 1]: opt(x) >= q + opt(r)
    => greedy(x) <= opt(x)
    => x < a[n - 1] + a[n - 2]

    => Thử hết các giá trị trên miền [2, a[n - 1] + a[n - 2] - 1] bằng 2 cách.
